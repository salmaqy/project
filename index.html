Production Diary: My Portfolio Progress Check

Entry 2

Date: 12 March 2026

  • What did you manage to create during this period?

    I made significant progress on styling my portfolio with CSS. I applied CSS Flexbox to lay out the navigation bar and the projects section, making them visually organised and easy to navigate. I also began adding responsive design using media queries so that the site adapts properly on mobile screens. In addition, I completed the CS Maths & Tools page with sections covering binary, hexadecimal, and logic gates.

  • What did you find most challenging?

    The most challenging aspect was getting the CSS Flexbox layout to behave consistently across different screen sizes. Understanding when to use flex-wrap, justify-content, and align-items together took a lot of trial and error. I also found writing clear explanations for the CS Maths topics difficult, as I wanted to be accurate without being overly technical.

  • What did you learn about web development or computer science maths from this work?

    I deepened my understanding of hexadecimal notation and how it is used as a shorthand for binary in computing contexts such as colour codes and memory addresses. I can now confidently convert between binary, decimal, and hexadecimal, which has helped me understand CSS colour values more intuitively.

  • What did you learn about web development or computer science?

    I learned how CSS media queries work and how to write mobile-first styles. I also explored CSS custom properties (variables) for maintaining a consistent colour palette across the site, which has made updating the design much easier.

  • What is one improvement you can guarantee for the upcoming weeks?

    I will add basic JavaScript interactivity to my portfolio — specifically a smooth-scroll navigation and a simple project filter feature — so that the site feels more dynamic and modern.
